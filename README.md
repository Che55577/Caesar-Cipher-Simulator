# 凱撒密碼加密程式

這是一個簡單的 **凱撒密碼 (Caesar Cipher) 加密程式**，它會將輸入的 **大寫英文字母** 向後偏移 3 個字母進行加密。  

例如：
- **HELLO** → **KHOOR**  
- **WORLD** → **ZRUOG**  

## 功能介紹

- 只能輸入 **大寫英文字母 (A-Z)**，不支援小寫字母或符號。
- 若輸入非大寫英文字母，則會顯示錯誤訊息。
- 加密方式使用 **Caesar Cipher**，即字母向後移動 3 位，若超過 `Z` 則會循環回 `A`。

---

## 程式運作方式

1. **使用者輸入字串**
2. **程式判斷是否為大寫英文字母**
3. **進行凱撒加密**
4. **輸出加密結果**
5. **若輸入包含非大寫英文字母，則顯示錯誤訊息**
